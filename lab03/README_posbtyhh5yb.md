Отчёт по программе «Симулятор лесных пожаров»
1. Назначение программы
Программа представляет собой симуляцию распространения лесного пожара на клеточном поле размером 100×100. Каждая клетка может находиться в одном из нескольких состояний: дерево, огонь, пепел, восстановление или вода. Симуляция выполняется в графическом интерфейсе с использованием библиотеки Tkinter и обновляется через равные интервалы времени.
2. Описание работы программы
Игровое поле состоит из сетки клеток. Каждая клетка имеет состояние и таймер. Состояние определяет внешний вид клетки и её поведение в симуляции. Таймер используется для отсчёта времени перехода клетки из одного состояния в другое.
Основные состояния клетки:
• Дерево – обычное состояние клетки.
• Огонь – клетка горит и может распространять огонь на соседние клетки.
• Пепел – клетка после сгорания.
• Восстановление – стадия роста нового дерева.
• Вода – препятствие для распространения огня.
3. Правила и механики симуляции
3.1 Ветер
Ветер влияет на вероятность распространения огня. Направление ветра выбирается случайно при запуске программы (влево, вправо, вверх, вниз или отсутствие ветра). Если огонь распространяется в направлении ветра, вероятность возгорания соседней клетки увеличивается на значение силы ветра.
3.2 Восстановление
После стадии пепла клетка переходит в состояние восстановления. На этой стадии происходит рост нового дерева. После истечения таймера восстановления клетка снова становится деревом.
3.3 Пепел
Когда дерево полностью сгорает, клетка переходит в состояние пепла. Пепел остаётся на карте определённое количество шагов симуляции. После завершения таймера пепел превращается в стадию восстановления.
3.4 Вода
Вода является препятствием для распространения огня. Огонь не может перейти на клетку воды. На карте изначально создаются два озера, а также пользователь может добавлять воду вручную.
3.5 Молния
Молния позволяет пользователю вызвать мгновенное возгорание в выбранной области. После клика по карте происходит удар молнии, который поджигает деревья в радиусе вокруг выбранной клетки.
3.6 Ядерный взрыв
Ядерный взрыв — специальная функция, которая мгновенно поджигает почти всё поле. Все клетки, кроме воды, переходят в состояние огня.
4. Управление программой
Панель управления позволяет:
• поджигать клетки;
• добавлять воду;
• вызывать удар молнии;
• запускать ядерный взрыв;
• восстанавливать весь лес;
• ставить симуляцию на паузу.
5. Статистика
В интерфейсе отображается статистика количества клеток каждого типа: деревьев, огня, воды и пепла. Это позволяет наблюдать динамику распространения пожара.
